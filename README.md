# SEO Friendly URLs with friendly_id:demo app/tutorial

## Gems
* friendly_id

## Versions
* Ruby: 3.0.1
* Rails:  6.1.4


## Running
1. Run bundler
```
bundle install
```
2. Run migrations
```
rails db:migrate
```
3. Start rails server
```
rails s
```

## Please Note (Windows Users Only)
un-comment this gem in your Gemfile, if you're using windows
